How it works?

  • Attack on availability

    A Distributed Denial of Service (DDoS) attack is an attempt to make an online service unavailable by overwhelming it with traffic from multiple sources. It overloads the server with fake packets so real users cannot access the resource.

  • Low Orbit Ion Cannon (LOIC)

    Low Orbit Ion Cannon is an open-source network stress testing and denial-of-service attack application, written in C#. When multiple users use this tool to attack the same url or ip address, the server becomes unresponsive due to the amount of packet requests it is trying to retreive and return a response. This tool was popularized a few years ago by the hacker group know as anonymous.

  • Victim Effects

    The host server becomes overloaded and unable to respond to the intended recipients, which affects the availability of the company's resource. This can cost a company an unknown amound of money in bandwidth, server repairs/shutdowns along with lost opportunity costs of sales & new clients.

  • Countermeasures

    There are no full-proof protections from a DDoS attack. Large companies can take a targetted server offline and replace it with a backup server to correct the traffic termporarily, which still costs money. Most companies end up taking a re-active approach by finding the attackers through tracing and attempting legal action as was the case with Anonymous.
